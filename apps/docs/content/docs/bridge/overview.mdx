## How Bridges Work

### Circle CCTP (Burn & Mint)
1. Burn USDC on source chain
2. Attestation service verifies burn
3. Mint USDC on destination chain
4. No wrapped tokens needed

### Soroban Bridge (Lock & Release)
1. Lock XLM in Stellar contract
2. Validators verify lock
3. Mint wXLM (wrapped XLM) on Ethereum
4. Reverse process to unlock

### Wormhole (Guardian Network)
1. Lock assets in source chain contract
2. Guardians observe and sign message
3. Message posted to destination chain
4. Assets released or minted
