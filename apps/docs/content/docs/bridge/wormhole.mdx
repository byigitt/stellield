# Wormhole Bridge

Decentralized cross-chain messaging protocol connecting 20+ blockchains including Stellar, Solana, and Ethereum.

## Overview

**Key Features**:
- ✅ **Multi-Chain**: 20+ supported blockchains
- ✅ **Guardian Network**: Decentralized validators
- ✅ **Fast**: 3-8 minute average
- ✅ **High Liquidity**: Billions in TVL

**Considerations**:
- ⚠️ Guardian trust assumptions
- ⚠️ Wrapped token complexity
- ⚠️ Higher gas costs than alternatives

## How It Works

1. **Lock** assets in source chain contract
2. **Guardians** observe and sign attestation (19 independent nodes)
3. **VAA** (Verifiable Action Approval) posted on-chain
4. **Claim** wrapped assets on destination chain

## Supported Routes (Stellar)

- Stellar ↔ Solana
- Stellar ↔ Ethereum
- Stellar ↔ Polygon
- Stellar ↔ BSC
- And 15+ other chains

## Wrapped Assets

**whXLM** (Wormhole XLM):
- Available on Solana, Ethereum, etc.
- Backed 1:1 by locked XLM
- Tradeable on destination DEXs

**Unwrapping**:
- Return whXLM to bridge
- Receive native XLM on Stellar
- Same fee structure

## Speed

**Average**: 3-8 minutes

**Breakdown**:
1. Source transaction: &lt;1 minute
2. Guardian signatures: 1-2 minutes
3. VAA availability: &lt;1 minute
4. Destination claim: 1-5 minutes

**Fastest Route**: Stellar → Solana (~2 minutes)
**Slowest Route**: Stellar → Ethereum (~15 minutes)

## Troubleshooting

**VAA Not Available?**
- Wait for guardian confirmations (1-2 min)
- Check Wormhole explorer

**Claim Failed?**
- Ensure gas on destination chain
- Retry claim transaction
- Check VAA expiry

**Wrapped Token Not Showing?**
- Add token contract to wallet
- Verify correct destination address
- Check Wormhole explorer status
